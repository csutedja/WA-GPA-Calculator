# Westford Academy GPA Calculator
_Easy to use Python program used for calculating a student's unweighted/weighted grade point average based on the <a href="https://www.westfordk12.us/sites/g/files/vyhlif5181/f/uploads/2223_school_profile.pdf">Westford Academy GPA system</a>._


**To use:**
1. Launch main.py. You will be presented with 3 options, calculate weighted GPA, unweighted GPA, and exit the program.
2. If you choose one of the first 2 options, the program will ask you how many courses you are taking (it will also ask what kind of course if you choose to calculate a weighted GPA).
3. Enter in your grades for each kind of course (CP/Honors/AP).
4. The program will output your GPA!
5. If you would like to repeat the program/end it, follow the on-screen instructions.
